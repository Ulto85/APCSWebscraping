Steps

First import the required modules

from bs4 import BeautifulSoup
import requests

Next we'll make a list of websites we want to scrape

new_sites= ["cnn","msnbc","nbcnews"]

Next we'll make a function that will get the top story

def getTopStory(site):
    scraper = BeautifulSoup(requests.get(f"https://www.{site}.com").content,"html.parser")
    return scraper.find_all('h2')[0].get_text()

Let's go line by line. First we use BeautifulSoup to create a scraper object. This is the object that will be used to scrape the website. Next we use requests to grab the content from a site using .content, and pass a parameter saying it's html so our scraper can process it propery. Next we use the scraper to .find_all() to find all the h2 tags. Finally we return the text of the first h2 tag. What is an h2 tag? If we inspect one of the websites we can see that the news stories are in h2 tags.



Now we'll write a function to scrape all sites in our list

for site in new_sites:
    print(f"Top Story from: {site} \n {getTopStory(site)} \n")
